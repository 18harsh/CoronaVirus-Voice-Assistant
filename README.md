# CoronaVirus-Voice-Assistant
A python voice assistant that you can ask corona-virus related questions

<b>like</b>:

How many cases are there in India

How many people died in Japan

active cases in Mainland China


# Requirements:
BeautifulSoup

requests

pandas

pyttsx3

speech_recognition

selenium
